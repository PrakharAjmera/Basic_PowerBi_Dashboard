# 📊 Data Professional Survey Dashboard (Power BI)

## 📌 Project Overview
This project analyzes a survey of data professionals using Power BI.  
The goal was to uncover insights about salaries, skills, and satisfaction levels.

## 🔍 Key Insights
- Data Scientists earn the highest salaries
- Python is the most preferred programming language
- Breaking into data is perceived as moderately difficult
- Work-life balance satisfaction is higher than salary satisfaction

## 🛠 Tools Used
- Power BI
- Power Query
- DAX

## 🚀 Learnings
- Built interactive dashboards
- Improved data storytelling
- Practiced real-world data analysis
